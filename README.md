# Sistema de Gerenciamento - Academia (Projeto Faculdade)

/// código por matheus augusto (tive que fazer sozinho professora, mas tudo bem)

O repositório do nosso projeto prático de programação! Este sistema foi desenvolvido em Python para automatizar a gestão de uma academia, aplicando conceitos fundamentais de estruturas de dados e modularização de código durante as aulas de programação.

O objetivo principal é gerenciar o cadastro de alunos, organizar as filas de atendimento na recepção e controlar o histórico de check-in de treinos de forma limpa e interativa no terminal com um menu prático.

///

## Como Rodar o Programa

1. Requisitos: Python instalado na versão 3.10 ou superior.
2. Biblioteca externa: Este projeto utiliza a biblioteca 'colorama' para estilizar o terminal. Instale a biblioteca rodando o comando no terminal:

   pip install colorama
